# Java 后端面试题（资深 / 架构师视角）

> 本套笔记在原始整理稿基础上，由资深 Java 技术专家 / 架构师视角**重写与完善**：去除重复、修正错误、补全原理与源码、更新到现代技术栈（**JDK 17/21、Spring Boot 3、Jakarta EE**），并强调「原理 → 源码 → 工程权衡 → 面试话术」的回答结构。
>
> 适用对象：3 年以上后端工程师备战中高级 / 架构师面试，亦可作为体系化复习手册。

## 阅读说明

- 每个专题独立成文，问题按「**一句话回答 → 展开原理 → 关键源码/图示 → 实战与坑 → 延伸追问**」组织，方便快速回顾，也方便面试时分层作答。
- 标注 `🔥` 的为高频核心题，标注 `⭐` 的为加分 / 区分度题，标注 `🆕` 的为相对新版本（JDK 9+/17+/21）的知识点。
- 代码示例尽量给出**可运行的最小片段**与**结论注释**，避免空泛描述。
- 版本基线：JDK 默认以 **LTS（8 / 17 / 21）** 对比讲解；Spring 以 **6.x / Spring Boot 3.x** 为主，必要处标注与旧版本差异。
