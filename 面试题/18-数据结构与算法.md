## 一、复杂度与数据结构

### 1. 时间/空间复杂度 ⭐

大 O 表示最坏增长趋势：`O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(2ⁿ) < O(n!)`。

### 2. 常见数据结构选型 🔥

| 结构 | 特点 | 查/增删 |
| --- | --- | --- |
| 数组 | 连续、随机访问 | 查 O(1) / 增删 O(n) |
| 链表 | 非连续、指针 | 查 O(n) / 增删 O(1) |
| 栈 | LIFO | — |
| 队列 | FIFO | — |
| 哈希表 | 散列定位 | 平均 O(1) |
| 二叉搜索树/红黑树 | 有序 | O(log n) |
| 堆（优先队列） | 取极值 O(1)、增删 O(log n) | TopK |
| 跳表 | 有序、范围查询 | O(log n)（Redis ZSet） |
| 图 | 关系网络 | BFS/DFS |
| Trie 前缀树 | 字符串前缀 | O(len) |

## 二、排序算法

### 3. 常见排序对比 🔥

| 算法 | 平均 | 最坏 | 空间 | 稳定 |
| --- | --- | --- | --- | --- |
| 冒泡 | O(n²) | O(n²) | O(1) | 稳定 |
| 选择 | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入 | O(n²) | O(n²) | O(1) | 稳定 |
| **快排** | O(n log n) | O(n²) | O(log n) | 不稳定 |
| **归并** | O(n log n) | O(n log n) | O(n) | 稳定 |
| **堆排** | O(n log n) | O(n log n) | O(1) | 不稳定 |
| 计数/桶/基数 | O(n+k) | — | O(n+k) | 稳定 |

> Java `Arrays.sort`：基本类型用**双轴快排**（不稳定，无需稳定性），对象数组用 **TimSort**（归并改进，稳定）。

### 4. 快速排序模板 ⭐

```java
void quickSort(int[] a, int l, int r) {
    if (l >= r) return;
    int pivot = a[l + (r - l) / 2], i = l, j = r;
    while (i <= j) {
        while (a[i] < pivot) i++;
        while (a[j] > pivot) j--;
        if (i <= j) { int t = a[i]; a[i] = a[j]; a[j] = t; i++; j--; }
    }
    quickSort(a, l, j);
    quickSort(a, i, r);
}
```

## 三、高频手撕题与思路

### 5. 高频题清单 🔥

- **数组/双指针**：两数之和、三数之和、接雨水、最长无重复子串、合并区间。
- **链表**：反转链表、判断环（快慢指针）、合并两个有序链表、找倒数第 K 个、LRU。
- **二分查找**：旋转数组找目标、找第一个/最后一个位置。
- **栈/队列**：有效括号、最小栈、单调栈（每日温度）、用栈实现队列。
- **二叉树**：各种遍历（递归/迭代/层序）、最大深度、最近公共祖先、判断对称、路径和。
- **动态规划**：爬楼梯、最大子数组和、最长递增子序列、背包、编辑距离、打家劫舍。
- **回溯**：全排列、子集、组合、N 皇后。
- **堆/TopK**：第 K 大、出现频率前 K、合并 K 个有序链表。
- **图**：岛屿数量（BFS/DFS）、拓扑排序。

### 6. 反转链表模板 ⭐

```java
ListNode reverse(ListNode head) {
    ListNode prev = null, cur = head;
    while (cur != null) {
        ListNode next = cur.next;
        cur.next = prev;
        prev = cur;
        cur = next;
    }
    return prev;
}
```

### 7. 二分查找模板 ⭐

```java
int binarySearch(int[] a, int target) {
    int l = 0, r = a.length - 1;
    while (l <= r) {
        int mid = l + (r - l) / 2;  // 防溢出
        if (a[mid] == target) return mid;
        else if (a[mid] < target) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}
```

### 8. LRU 实现思路 🔥

`HashMap + 双向链表`：Map 提供 O(1) 查找，双向链表维护访问顺序（最近访问移到头部，淘汰尾部）。Java 可直接用 `LinkedHashMap`（`accessOrder=true` + 重写 `removeEldestEntry`）。

### 9. TopK 思路 ⭐

- 小顶堆（大小为 K）：遍历维护，O(n log k)，适合海量数据。
- 快速选择（快排 partition 思想）：平均 O(n)。

## 高频追问清单

- 快排为什么不稳定、最坏 O(n²) 怎么优化？→ 随机基准 / 三数取中。
- 判断链表有环？→ 快慢指针（Floyd）。
- LRU 怎么 O(1)？→ 哈希表 + 双向链表（第 8 题）。
- 海量数据找 TopK？→ 小顶堆 / 分治（第 9 题）。
